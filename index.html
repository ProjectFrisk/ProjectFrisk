Project Frisk

Experience seamless iOS sideloading.

DNS Setup

Install the Project Frisk DNS profile to block Apple's revoke servers and keep your apps running.

How to Install

  1. Click the button above to download the file from MediaFire.
  2. Tap Allow when your device prompts you.
  3. Open your device Settings app.
  4. Tap Profile Downloaded near the top.
  5. Tap Install in the top right corner.
