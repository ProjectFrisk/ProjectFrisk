Project Frisk

Experience seamless iOS sideloading.

Step 1: DNS Setup

To ensure successful sideloading, you need to install the Project Frisk DNS profile. This will block Apple's revoke servers.

Instructions

  1. Tap Allow when prompted to download.
  2. Open your device Settings app.
  3. Tap Profile Downloaded near the top.
  4. Tap Install in the top right corner.
