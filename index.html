How to Install

  1. Click the configuration button above to open MediaFire.
  2. Tap Allow when the system warning drops.
  3. Open your native iOS Settings dashboard.
  4. Tap Profile Downloaded near the header row.
  5. Press Install in the top-right terminal corner.
